To upload the code to the board:

1) open arduino IDE on a computer
2) make sure that it has the due board installed
    - go to Tools -> Boards -> Boards Manager
    - search for Due install(if it is not)
3) under Tools select the com port the board is on
4) under Tools -> boards select the Due(programming port)
5) push the upload button ( it is round and has an arrow on it)
    - or push control-Shift-U
6) to see the data open the serial monitor or serial plotter under Tools
